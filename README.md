# ValuStock — แพลตฟอร์มประเมินมูลค่าหุ้นสำหรับนักลงทุนไทย

เครื่องมือประเมินมูลค่าหุ้นระดับมืออาชีพ สร้างด้วย **Next.js 15 (App Router)** + **React 19** + **TypeScript** + **Tailwind CSS** พร้อม **ระบบสมาชิก/แพ็กเกจ** ราคาเป็นเงินบาท ออกแบบสำหรับนักลงทุนไทย

> ⚠️ **ข้อมูลในเดโมเป็นตัวอย่างเพื่อการสาธิตเท่านั้น ไม่ใช่ข้อมูลเรียลไทม์และไม่ใช่คำแนะนำการลงทุน** การลงทุนมีความเสี่ยง ผู้ลงทุนควรศึกษาข้อมูลจริงก่อนตัดสินใจ

---

## ✨ ฟีเจอร์หลัก

- **ประเมินมูลค่าหลายโมเดล** — DCF (กระแสเงินสดคิดลด 2 ระยะ), Graham Number และมูลค่าเหมาะสมแบบถ่วงน้ำหนัก พร้อม Margin of Safety และสัญญาณ "ราคาต่ำกว่า/เหมาะสม/สูงกว่ามูลค่า"
- **เครื่องคำนวณ DCF แบบอินเทอร์แอกทีฟ** — ปรับอัตราเติบโต FCF, WACC, อัตราโตระยะยาว และจำนวนปี แล้วเห็นมูลค่าที่แท้จริงทันที
- **อัตราส่วนการเงินครบ** — P/E, P/BV, P/S, EV/EBITDA, ROE, อัตราปันผล, PEG, อัตรากำไรสุทธิ
- **สกรีนเนอร์คัดกรองหุ้น** — ค้นหา/กรองตามกลุ่มอุตสาหกรรม คัดเฉพาะหุ้นราคาต่ำกว่ามูลค่า เรียงตาม Margin of Safety / P/E
- **เปรียบเทียบหุ้น** — วางได้สูงสุด 4 ตัวแบบเคียงข้าง
- **รายการโปรด (Watchlist)** — ติดตามหุ้นที่สนใจ
- **ระบบสมาชิก 3 ระดับ** — พร้อมการจำกัดสิทธิ์ตามแพ็กเกจ และหน้าชำระเงินจำลอง (พร้อมเพย์/บัตรเครดิต/โอน)
- **ธีมสว่าง/มืด** + รองรับมือถือและเดสก์ท็อป

## 💎 แพ็กเกจสมาชิก

| แพ็กเกจ | ราคา | สิทธิ์เด่น |
|---------|------|-----------|
| **ฟรี** | 0 ฿ | ดูหุ้น 5 ตัว, อัตราส่วนพื้นฐาน, รายการโปรด 3 ตัว |
| **โปร** | 299 ฿/เดือน (2,990 ฿/ปี) | ดูหุ้นทั้งตลาด, เครื่องคำนวณ DCF, สกรีนเนอร์, รายการโปรดไม่จำกัด |
| **พรีเมียม** | 599 ฿/เดือน (5,990 ฿/ปี) | ทุกอย่างในโปร + เปรียบเทียบหุ้น, แจ้งเตือน, ส่งออก CSV |

แก้ไขแพ็กเกจ ราคา และสิทธิ์ได้ที่ `lib/plans.ts`

## 🚀 เริ่มใช้งาน

ต้องมี **Node.js 18.18+** (แนะนำ 20 หรือ 22)

```bash
npm install
npm run dev
```

เปิดเบราว์เซอร์ที่ http://localhost:3000

คำสั่งอื่น:

```bash
npm run build   # สร้างเวอร์ชัน production
npm run start   # รันเวอร์ชัน production
```

> หมายเหตุ: โปรเจกต์ใช้ Google Fonts (Bricolage Grotesque, IBM Plex Sans Thai, IBM Plex Mono) ผ่าน `next/font` — เครื่องของคุณต้องต่ออินเทอร์เน็ตตอน build ครั้งแรกเพื่อดาวน์โหลดฟอนต์

## 🔌 การเชื่อมต่อ API ข้อมูลจริง

ปัจจุบันข้อมูลหุ้นมาจาก `lib/stocks.ts` (ตัวอย่าง) ออกแบบให้สลับเป็น API จริงได้ง่าย โดยแก้ให้ `getStock()` / `STOCKS` ดึงจากแหล่งข้อมูลแทน แนะนำ:

- **Financial Modeling Prep (FMP)** — ครบที่สุดสำหรับประเมินมูลค่า มี endpoint งบการเงิน อัตราส่วน และ DCF สำเร็จรูป
- **Alpha Vantage** — ฟรีเทียร์ใจกว้าง มีงบการเงินย้อนหลังยาว เหมาะเริ่มต้น
- **EODHD** — ครอบคลุม **ตลาดหุ้นไทย (SET)** ใช้ ticker ลงท้าย `.BK` (เช่น `PTT.BK`)
- **Settrade Open API** — ทางการของไทย เน้นราคาเรียลไทม์/ส่งคำสั่งซื้อขาย (ต้องเปิดบัญชีกับโบรกเกอร์)

ตัวอย่างแนวทาง: สร้างไฟล์ `lib/api.ts` ที่ map ผลลัพธ์จาก API ให้ตรงกับ type `Stock` ใน `lib/types.ts` แล้วตรรกะประเมินมูลค่าทั้งหมดใน `valuation.ts` จะทำงานต่อได้ทันที

## 💳 ระบบชำระเงิน (ต่อยอด)

หน้าชำระเงินปัจจุบันเป็น **การจำลอง** (ไม่มีการตัดเงินจริง) สำหรับใช้งานจริงในไทยแนะนำเชื่อมต่อ:

- **Omise / Opn** — รองรับบัตรเครดิตและพร้อมเพย์ เป็นที่นิยมในไทย
- **GB Prime Pay**, **2C2P** — ผู้ให้บริการชำระเงินในไทย
- **Stripe** — สำหรับลูกค้าต่างประเทศ

จุดเชื่อมต่ออยู่ที่ฟังก์ชัน `confirm()` ในหน้า `app/pricing/page.tsx`

## 🔐 ระบบสมาชิก (ต่อยอด)

ระบบ auth ปัจจุบันเป็นการจำลองเก็บใน localStorage แนะนำต่อยอดด้วย **NextAuth.js (Auth.js)** + ฐานข้อมูล (เช่น PostgreSQL/Prisma) เพื่อใช้งานจริง
